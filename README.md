# 説明
二次元配列を変数宣言用のテキストデータに変換して、クリップボードにコピーする

## 活用例
VBAにて一発で配列を定義するコードを簡単に書ける


# 使い方
実行サンプル「Sample_ClipCopyArray2D.xlsm」の中は以下のようになっている。

使用例
